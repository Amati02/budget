# Product Overview

## Budget Tracker

A personal finance management application for tracking household expenses and income. The system helps users monitor their budget, categorize transactions, and visualize spending patterns.

## Key Features

- **Expense & Income Tracking**: Record and categorize financial transactions
- **Budget Management**: Set monthly budgets and track spending against targets
- **Visual Analytics**: Charts and reports showing spending patterns by category
- **Telegram Bot Integration**: Add transactions via Telegram for convenience
- **Multi-language Support**: Russian language interface (primary)

## Target Users

Personal/household use for budget-conscious individuals or couples managing shared finances.

## Data Storage

Uses JSON file-based storage (`budget_data.json`) for simplicity and portability.
